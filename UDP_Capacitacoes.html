Conexão com Google Sheets
Para conectar o sistema à planilha, você precisa de uma chave de API do Google e do ID da sua planilha. Veja o guia de configuração abaixo.
O ID está na URL da planilha: docs.google.com/spreadsheets/d/[ID AQUI]/edit
Google Cloud Console → APIs → Credentials → API Key. Habilite a Google Sheets API.
  1. Acesse console.cloud.google.com
  2. Crie um projeto (ou use um existente)
  3. Vá em APIs e serviços → Biblioteca → busque Google Sheets API → Ativar
  4. Vá em Credenciais → Criar credencial → Chave de API
  5. Copie a chave e cole no campo acima
  6. Na planilha, clique em Compartilhar → qualquer pessoa com o link pode visualizar
  7. Copie o ID da planilha da URL e cole acima
